# INTERCAL for Visual Studio Code

Syntax highlighting for INTERCAL, the Computer Language with No Pronounceable Acronym.

## Features

- Syntax highlighting for `.i` files
- All classic INTERCAL keywords: DO, PLEASE, GIVE UP, READ OUT, WRITE IN, COME FROM, etc.
- Variable types: spot (`.`), two-spot (`:`), four-spot (`::`), tail (`,`), hybrid (`;`), double-hybrid (`;;`), and cat box (`[]`)
- Operators: mingle (`$`), select (`~`), double worm (`=`), unary (`&`, `V`, `?`)
- Constants: mesh (`#`), double mesh (`##`), quad mesh (`####`)
- Gerunds: NEXTING, CALCULATING, BOXING, FEEDING, PETTING, etc.
- Labels: `(100)`
- Splatted statements highlighted as errors

## Installation

1. Copy the `vscode-intercal` folder to `~/.vscode/extensions/`
2. Restart VS Code
3. Open any `.i` file
