# Learn INTERCAL

Tiny programs demonstrating each language feature. Open them in VS Code, set a breakpoint on the first line, and step through.

| # | File | Concept |
|---|------|---------|
| 01 | hello.i | Hello World — Turing tape output |
| 02 | variables.i | Spots, two-spots, four-spots, constants |
| 03 | mingle.i | Mingle ($) — bit interleaving |
| 04 | select.i | Select (~) — bit extraction |
| 05 | unary.i | Unary & V ? — adjacent bit operations |
| 06 | calculate.i | Assignment and nested expressions |
| 07 | next-resume.i | Subroutine calls |
| 08 | forget.i | Dropping the return address |
| 09 | abstain-reinstate.i | Disabling and re-enabling statements |
| 10 | come-from.i | The opposite of GOTO |
| 11 | stash-retrieve.i | Saving and restoring variables |
| 12 | ignore-remember.i | Read-only variables |
| 13 | please.i | Politeness requirements |
| 14 | arrays.i | Tails, hybrids, and SUB indexing |
| 15 | syslib.i | Arithmetic via the system library |
| 16 | write-in.i | User input |
| 17 | come-from-loop.i | Looping with COME FROM |
